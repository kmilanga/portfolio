# Portfolio Profesional | Estrategia & Desarrollo Web

[![Netlify Status](https://api.netlify.com/api/v1/badges/TU_API_ID/deploy-status)](https://tu-url.netlify.app)

Este es mi portfolio personal, diseñado y programado desde cero. No es solo una muestra de código, sino una herramienta de conversión orientada al marketing digital y la experiencia de usuario (UX).

## 🚀 Enfoque Estratégico
A diferencia de un portfolio convencional, este proyecto se basa en el **Enfoque C2**:
* **Rendimiento Extremo:** Optimización de Core Web Vitals para una carga instantánea.
* **Diseño Minimalista:** Foco en la legibilidad y la esencia de marca.
* **Estrategia de Conversión:** Estructura pensada para guiar al usuario desde la propuesta de valor hasta el contacto.

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5 Semántico, CSS3 (Mobile-First), JavaScript Vanilla.
* **Despliegue:** Netlify con Integración Continua (CI/CD).
* **Herramientas:** Adobe Suite / Figma para el diseño visual.
